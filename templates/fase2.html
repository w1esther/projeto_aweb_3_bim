FASE 2: INTRODUÇÃO AO CSS

O que é o CSS?


O CSS (Cascading Style Sheets) é a linguagem responsável por definir o estilo e a aparência das páginas da web. Com ele, é possível controlar cores, fontes, tamanhos de texto, margens, espaçamentos, posicionamentos e até criar efeitos visuais e animações. O CSS trabalha junto com o HTML, que estrutura o conteúdo da página, enquanto o CSS cuida do design e da formatação. Dessa forma, os sites se tornam mais bonitos, organizados e agradáveis de usar, além de poderem se adaptar a diferentes tamanhos de tela, como computadores, tablets e celulares.



As três formas de incorporar o CSS:


CSS Inline: É aplicado diretamente dentro das tags HTML, usando o atributo style. Serve para formatar apenas um elemento específico.
Vantagem: rápido para ajustes simples.
Desvantagem: difícil de manter se o site tiver muitos estilos.



CSS Interno: Fica dentro da tag style no cabeçalho head do HTML. É útil quando o estilo é usado somente naquela página.
Vantagem: tudo está no mesmo arquivo.
Desvantagem: não é prático para sites com várias páginas.



CSS Externo: O estilo é colocado em um arquivo separado (ex: style.css) e ligado ao HTML com a tag link.
Vantagem: facilita a manutenção e o reaproveitamento dos estilos.
Desvantagem: precisa carregar um arquivo a mais, mas é o método mais usado em sites reais.




Como declarar uma propiedade no CSS:

Para declarar uma propriedade no CSS, é preciso seguir uma estrutura simples que usa seletores, propriedades e valores. Primeiro, escolhe-se o elemento HTML que será estilizado (como p, h1 ou body). Depois, dentro de chaves {}, escrevemos a propriedade e o valor desejado, separados por dois pontos : e finalizados com ponto e vírgula ;.



Nesse caso, o seletor p indica que o estilo se aplica a todos os parágrafos. A propriedade color define a cor do texto, e o valor blue é a cor escolhida. Já font-size define o tamanho da fonte, com o valor 16px.
