Pincipais tags HTML

Imagens

A tag <img> é usada para inserir imagens em uma página HTML. Ela precisa do atributo src para indicar o arquivo da immagem e alt para descrever o conteúdo caso ela nao consiga.

Tabelas

As tabelas em HTML são criadas com a tag <table> e organizadas em linhas (<tr>) e colunas (<td>), servindo para exibir dados de forma estruturada, como planilhas e horários.

Formulários

Os formulários (<form>) permitem coletar dados do usuário, como nome ou e-mail. Utilizam campos como <input>,<textarea> e <button> para enviar as informações.
