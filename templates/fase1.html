FASE 1: INTRODUÇÃO AO HTML


O que é o HTML?


HTML é a linguagem de marcação usada para estruturar páginas da web. Com ele é possível organizar textos, imagens, links, tabelas e outros elementos que formam o conteúdo de um site. Ele funciona como a base da página, permitindo que navegadores interpretem e exibam as informações de forma compreensível para o usuário.


Como se organiza o HTML?


O HTML se organiza por meio de tags, que são marcas usadas para indicar a função de cada parte do conteúdo. Essas tags normalmente vêm em pares, com uma de abertura e outra de fechamento, e dentro delas fica o conteúdo. A estrutura básica de um documento HTML começa com a definição do tipo de arquivo, seguida das partes principais: o head, que guarda informações sobre a página, e o body, onde ficam os elementos que aparecem para o usuário no navegador.


As tags no HTML


As tags no HTML são comandos usados para marcar e organizar os elementos de uma página. Elas indicam ao navegador como cada parte do conteúdo deve ser exibida. Por exemplo, a tag <p> define um parágrafo, a <h1> a <h6> marcam títulos de diferentes níveis, a <a> cria links e a <img> insere imagens. A maioria das tags possui uma abertura <tag> e um fechamento </tag>, envolvendo o conteúdo, mas algumas, como <br> e <img>, são usadas sozinhas. Assim, as tags são a base da construção de qualquer site em HTML.


PRINCIPAIS TAGS HTML
html Define o início e o fim do documento HTML.
head Contém informações sobre a página (título, meta dados, links de estilos, scripts).
title Define o título que aparece na aba do navegador.
body Parte visível da página, onde ficam os conteúdos.
h1 a h6 Títulos e subtítulos, do mais importante (h1) ao menos importante (h6).
p Define um parágrafo de texto.
a Cria links para outras páginas ou seções.
img Insere imagens.
br Quebra de linha.
ul Lista não ordenada (com marcadores).
ol Lista ordenada (com números).
li Item de lista.
table Cria tabelas.
tr Linha da tabela.
td Célula de tabela.
th Cabeçalho da tabela.
div Seção ou bloco de conteúdo.

Os atributos


Os atributos no HTML são usados para complementar as tags, adicionando informações ou ajustando a forma como os elementos aparecem e funcionam na página. Eles sempre são escritos dentro da tag de abertura, no formato nome="valor". Por exemplo: Na tag <a>, o atributo href indica o endereço para onde o link vai direcionar. Na tag <img>, o atributo src define o caminho da imagem, enquanto alt descreve a imagem caso ela não carregue. Atributos como id e class ajudam a identificar e organizar os elementos, sendo muito usados junto com o CSS e JavaScript. Com os atributos, o HTML se torna mais flexível, permitindo que os elementos tenham características e comportamentos específicos.
